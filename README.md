# WhatsBot

A automated script for send messages/media to multiple contacts on WhatsApp Messenger
Um script automatizado para enviar mensagens/mídias para múltiplos contatos no WhatsApp Messenger

##How to use
##Como usar

<ul>
  <li>Install Python 3.9 or superior
    <br>
    Instale o Python 3.9 ou superior</li>
  <li>Execute the command in prompt/terminal:
    <br>
    Execute o comando no prompt/terminal
    <br>
    pip install -r requirements.txt (Windows) / pip3 install -r requirements.txt (Linux)</li>
  <li>In the folder, open the file "contacts.txt" and place the list of contacts you need to send messages/media. Obey this format: +(country code)(regional code)number. Example: +5522999887766. Do not use spaces or dashes.<br>Na pasta, abra o arquivo "contacts.txt" e coloque a lista de contatos que você precisa para enviar mensagens/mídia. Obedeça esse formato: +<código do país><código regional>número. Exemplo: +5522999887766. Não use espaços nem traços.</li>
    <li>Adjust your preferences in file "CONFIGURAÇÕES.txt" / Ajuste suas preferências no arquivo "CONFIGURAÇÕES.txt"</li>
    <li>If you send image/video, put your image/video in folder "img" with exact name "imagem". Example: imagem.jpeg
      <br>
    Se enviar imagem/vídeo, coloque sua imagem/vídeo na pasta "img" com o nome exato "imagem". Exemplo: imagem.jpeg</li>
